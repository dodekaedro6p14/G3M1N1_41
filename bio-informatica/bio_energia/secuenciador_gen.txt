Por qué esta herramienta es clave para tu proyecto:

Identificación de Especies: Si encuentras un hongo con aplicaciones útiles en la naturaleza, 
el primer paso es secuenciar su gen 18S rRNA o ITS. Este script te permite compararlo con la 
base de datos global para saber exactamente qué especie es.

Verificación de Modificación: Tras usar CRISPR-Cas9, puedes alinear la secuencia editada 
contra la secuencia original (Wild Type) para confirmar que la inserción o mutación ocurrió 
en el locus correcto.

Análisis de Dominios: Permite identificar si las enzimas fúngicas (como las celulasas para 
energía) tienen dominios conservados que puedan ser optimizados mediante biología sintética.

Limitación técnica importante: Debido a las políticas de seguridad de los navegadores (CORS), 
las peticiones directas desde el navegador a la NCBI pueden ser bloqueadas. Para que este 
código funcione en producción, deberás ejecutarlo a través de un pequeño servidor proxy 
(como un script de Python con Flask o Node.js con Express) que haga de intermediario.
